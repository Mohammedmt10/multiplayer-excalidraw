# Multiplayer Excalidraw

A real-time collaborative whiteboard application inspired by Excalidraw.  
Multiple users can join a shared room and draw on the same canvas with real-time synchronization.

---

## Features

- Real-time multiplayer canvas
- Room-based collaboration
- Shape drawing and freehand tools
- WebSocket-based synchronization
- Monorepo architecture

---

## Tech Stack

- Frontend: Next.js, HTML Canvas
- Backend: Node.js, Express, WebSockets
- Language: TypeScript
- Database: PostgreSQL
- Monorepo Tooling: Turborepo
- Package Manager: pnpm
